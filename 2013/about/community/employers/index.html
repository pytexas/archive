Texas Python Employers

Python job-seekers might be interested to know which potential employers use Python, and how they use it. This page can be used to take notes about what is known about local employers using Python. This list should be kept in alphabetical order by city and by company name within each city. See also the Python jobs board.

Austin area

  • BazaarVoice employs Python developers and sent swag (coozies) to PyTexas 2010

  • Cisco Systems
  • Dell has been known to hire local Python developers
  • Enthought employs Python developers and hosts Austin Python user group meetings at their office.

  • Journyx

  • Kforce (consulting/recruiting services)
  • Luna Data Solutions

  • Mutual Mobile employs Python/Django developers

  • Rackspace has an office in Austin and was a major sponsor of PyTexas 2010.

  • Resource Technical Management (recruiter)

  • Texas Water Development Board

Dallas/Fort Worth area

  • Gifts and Talents, LLC (Plano, Texas, USA)

  • Fish Technologies in Addison, TX regularly hosts the DFW Python meetings.

  • PERFECT-TIMING.BIZ -- Currently seeking experienced Python developer for substantial upgrades to a few different web applications primarily written in Python. please contact matt@perfect-timing.biz

  • Texas Instruments, Inc. -- Martin Taylor is the Senior Test Automation Engineer in the Engineering Services group at TI's Educational and Productivity Solutions (E&PS aka 'Calculator') division. His team has developed a test automation framework in Python and has automated close to 10,000 tests in Python. Martin's group has contract openings for Python programmers with additional skills in one or more of the following: wxPython, SQL (SQLAlchemy in particular), Java, Eclipse (from a plug-in developer perpective), embedded systems and math teaching. You can contact Martin Taylor as cmtaylor at ti dot com.

  • Systemware -> http://www.systemware.com/ -- Professional Services division, in Addison, seems to use Python.

  • Tellurian, Inc. -- Manufacturing & Distributor Of Custom & Off The Shelf Automated Process Equipment Including Soldering, Microflame Soldering, Brazing, Screwdriving, Dispensing, Vision Systems, & Robotics. Located in Grand Prairie.

  • Currency Technics & Metrics -- Located in Irving, is starting to ramp up a Python team to develop its next generation of software. You can contact Ron Smith as ron.smith at ctm dot com.

  • Vertis Communications -- premier provider of targeted advertising and marketing solutions to America's leading retail and consumer services companies. Looking for a Python programmer (job description -> http://www.python.org/community/jobs/#vertis-communications-dallas-tx-usa). Located in Irving (DFW airport side of Las Colinas).

  • ZeOmega, in Frisco, TX uses Python and Zope extensively in their products, and sponsored PyTexas 2010 significantly by allowing employee BradAllen to organize the conference during work hours. Several of the local DFW Python user group members are employed by ZeOmega. ZeOmega is a sponsor member of the Python Software Foundation starting in 2003 and has sponsored PyCon since 2003.

  • Qualbe Marketing Group -- Located just west of Downtown Fort Worth. Currently seeking a Python programmer for maintenance and development of external Python/Django web applications and internal Python CRM application. Please contact jobs at qualbe dot com.

  • Rice University Web Services (Houston, TX, USA)

  • Mouth Watering Media (Houston, TX, USA)

  • NASA 

San Antonio

  • Rackspace is based in San Antonio and was a major sponsor of PyTexas 2010.
