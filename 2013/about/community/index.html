Texas Python Community

The Texas Python Community offers many ways for you to get involved, help out, learn, and have fun.

Follow Announcements and Discussion

  • Follow PyTexas on Twitter primarily for announcements

  • Follow #PyTexas hashtag on Twitter for PyTexas-related community commentary

  • Subscribe to the The Texas Python mailing list to follow both discussion and announcements

Join a Local User Group

The Texas area has several user groups:

  • Austin User Group meets on the 2nd Wednesdays

  • Austin Web Python User's Group meets on the 4th Thursdays

  • The DFW Pythoneers usually meets in the North Dallas/Richardson area. There is also a DFW Python wiki.

  • PyHou - Houston Python Enthusiasts

If you're interested in starting a user group in another city, please raise the issue on the Texas Python mailing list.

Work

Check out the Texas Python Employers page for a list of potential job opportunities in the local area.

If you are company which has Python developer job openings, please feel free to post to one of the user group mailing lists above, or to the statewide Texas Python mailing list. 

Another great option for employers wishing to become visible to the community is to sponsor the PyTexas conference. 

Benefits

Those of us who enjoy Python programming, or just find it a practical tool, can benefit from knowing one another. Python is a kind of "gateway drug" into many technologies and there are always new things to learn. Here are a few benefits of creating an active local community:

  • professional development
  • knowledge sharing
  • great conferences like PyTexas
  • interesting people with which to socialize
  • a thriving local community gives employers confidence to use Python
